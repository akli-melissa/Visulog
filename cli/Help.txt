Voici les options que vous pouvez utiliser (VEUILLEZ BIEN RESPECTER LA SYNTAXE IMPOSÉE):

	--addPlugin = argument
	
	--justSaveConfigFile = nomDufichierDeConfiguration : argument1,argument2,...
	
	--loadConfigFile = nomDufichierDeConfiguration
	
"argument" :  peut avoir les valeurs suivantes :
	**-All : pour executer toutes les options en meme temps.
	
	**-countLinesPerAuthor : pour le nombre de lignes modifiées par un utilisateur.
	
	**-countLines : pour le nombre de lignes modifiées (Tous les utilisateurs).
	
	**-countMergeCommits :  pour le nombre de merges.
	
	**-countCommits : pour le nombre de commits.
	
	**-countCommitsPerDayOfWeek : pour le nombre de commits par jour de semaine ( Lundi,Mardi,...).
	
	**-countCommitsPerDayOfMonth : pour le nombre de commits par jours de mois ( 20 janvier,10 avril ,...).
	
	**-countCommitsPerHourOfDay : pour le nombre de commits par heures ( 13h,08h,...).
	**-userStats/user   : remplacer user avec le nom de l'auteur en remplacent les espaces ' ' avec '_'

"nomDufichierDeConfiguration" : le nom du fichier dans le quel vous voulez sauvegarder 
				 vos options argument1 argument2 separés par des virgules ','
